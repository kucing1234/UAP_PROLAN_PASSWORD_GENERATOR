# Secure Pass Manager (Java Swing)

Secure Pass Manager adalah aplikasi desktop berbasis Java Swing yang digunakan
untuk mengelola data akun dan password secara aman dan terstruktur. Aplikasi ini
mendukung fitur CRUD (Create, Read, Update, Delete) serta penyimpanan data
secara permanen menggunakan file CSV.

- Generate password secara acak dengan panjang yang ditentukan pengguna
- CRUD data password (Tambah, Lihat, Edit, Hapus)
- Penyimpanan data permanen menggunakan file CSV
- Sorting dan searching data password
- Copy password ke clipboard dengan double-click
- Antarmuka GUI responsif menggunakan Java Swing

- Java SE
- Java Swing (GUI)
- File Handling (CSV)
- OOP (Inheritance, Encapsulation, Polymorphism)
- Layout Manager (CardLayout, GridBagLayout, BorderLayout)
